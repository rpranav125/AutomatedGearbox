# Automated Gearbox
Designed and manufactured a high-torque 5:1 gear reduction system using CAD modeling, 3D printed assembly, and Arduino automation.

Key Features
Mechanical Design: Developed a custom gear profile in CAD (Fusion 360), optimizing for 3D printing tolerances and tooth engagement.

Additive Manufacturing: Iterated on 3D-printed prototype to minimize friction and ensure structural integrity under load.

Automation & Control: Integrated an Arduino microcontroller with NEMA 17 stepper motor and A4988 driver to automate rotation sequences and test the gearbox's precision.

Performance: Achieved 5:1 reduction, significantly increasing output torque for small-scale mechanical applications.

Future Directions: Minimize gear movement along shaft during motion. Test with gear reduction with wheel attachment.
